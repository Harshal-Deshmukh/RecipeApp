# 🍽️ RecipeApp

An Android app that lets you explore food categories and recipes.

## ✨ Features
- Browse food categories
- View meals by category
- Detailed recipe with ingredients & instructions
- Smooth image loading

## 🛠️ Tech Stack
- Kotlin + Jetpack Compose
- Retrofit
- Coil
- Navigation Component
- MVVM Architecture
- TheMealDB API

## 🚀 Getting Started
1. Clone the repo
2.  Open in Android Studio
3. Run the app

## 📡 API
This app uses [TheMealDB](https://www.themealdb.com/) free API.
